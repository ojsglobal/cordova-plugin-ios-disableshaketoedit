# cordova-plugin-ios-disableshaketoedit

## Description
Phongap Build plugin to disable the "Shake to undo" feature by setting the application's `applicationSupportsShakeToEdit` property to `NO`.

I needed a Phonegap Build version of the plugin here https://github.com/nleclerc/cordova-plugin-ios-disableshaketoedit so I created this repo.
